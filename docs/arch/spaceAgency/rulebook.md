## Scope

Space Agencies is an optional expansion module for SETI. When the expansion is used, some components are added to the base game immediately, while organizations and quick start cards form a separate optional way to play.

This Markdown document preserves the rule meaning of the PDF. Illustrations are not reproduced, but rule-bearing icons and component images are transcribed as text.

### Option Boundary

If the Space Agencies option is not selected, use the base-game rules and components only.

If the Space Agencies option is selected, the expansion's main deck cards, new alien species, and signal tokens are added to the base game as described below. Organizations and quick start cards are then an additional module within the expansion: use them together when playing with organizations, or leave them out for a base-style game with the expanded card and alien pool.

## Cover Premise

Humanity's search for alien life has grown more ambitious. With discovery within reach, powerful organizations have stepped in, each bringing its own vision, methods, and motives. Whether public, private, or fringe, these groups are reshaping how the search unfolds.

Will your organization be the one to make first contact?

## Using This Expansion

Add these components to the base game immediately, regardless of whether you use other parts of the expansion:

- More cards: Shuffle the 42 new cards into the main deck.
- Alien species: Mix the 3 new aliens with the original alien species. Randomly select 2 alien species for each game as usual.
- Signal tokens: Some new effects award signal tokens. Keep them within easy reach.

Organizations introduce a new way to play SETI and are used together with quick start cards.

## Base Game Delta Summary

This expansion is incremental to the base game:

- The base game normally lasts 5 rounds. With organizations, the game lasts 4 rounds: play only rounds 2, 3, 4, and 5.
- The base game uses starting income cards. With organizations, leave the starting income cards in the box; organization setup replaces them.
- The base game starts players at 4 publicity. With organizations, set all players to 0 publicity during setup, then resolve quick start cards and organization starting resources/effects.
- The base game prepares end-of-round card stacks for rounds 1-4. With organizations, prepare end-of-round cards only for rounds 2, 3, and 4.
- Base scan rules still apply. Signal tokens only add an extra way to mark signals during a scan action.
- Base alien setup still applies. The expansion increases the alien pool before randomly selecting 2 species.

## Playing With Organizations

Each player controls an organization with unique powers and an asymmetric starting setup and income.

Organizations let players achieve more than usual. To balance this and keep the game length under control, games with organizations last 4 rounds. Play only rounds 2, 3, 4, and 5.

### Setup With Organizations

1. Prepare end-of-round cards for rounds 2, 3, and 4 only.
2. Determine the starting player and set starting scores based on turn order as usual.
3. Set publicity to 0 for all players.
4. Deal each player:
   - 2 random organizations.
   - 3 random quick start cards.
   - 4 random cards from the main deck.
5. Each player chooses 1 organization and 2 quick start cards.
6. Each player keeps all 4 main deck cards.
7. All players simultaneously resolve their 2 quick start cards.
8. All players gain all starting resources and effects of their chosen organization.

Knowing the 4 main deck cards you start with can help you choose your organization and quick start cards.

### Quick Start Cards

The top half of a quick start card may instruct you to place an orbiter, trace, or signals. Mark those, but do not gain any effects or other benefits for doing so.

The bottom half may give starting resources, points, and other effects. These are usually related to what was marked above, but not always.

After resolving quick start cards, return them to the box unless they were tucked for income.

### Organization Card Anatomy

Organizations may contain these areas:

- Starting resources and effects: Resolve these during setup after quick start card selection.
- Passive power: Ongoing ability text on the organization.
- Once per round ability: A free action available once per round. When used, mark its space as a reminder. Remove the marker when resolving income.
- Income increases: Most organizations let you tuck cards for income. Gain the resource from each tucked card immediately.
- Income: Do not resolve income at game start. Resolve income at the start of rounds 3, 4, and 5.

Income increases are resolved after taking all starting resources. You may choose which cards to tuck from the 4 main deck cards dealt in setup and any extra cards gained from the organization or quick start cards.

Resolve multiple income increases one by one. If one tucked card gives you a random card, you can use that exact card for a following income increase.

### First Game Recommendation

When playing with organizations for the first time, consider starting with organizations numbered 1-4, shown in the bottom-left corner of the organization cards. These four organizations are generally less complex.

Recommended first organization game setup:

- Give everyone 1 of organizations 1-4.
- Give everyone 2 quick start cards instead of 3.
- Give everyone 4 cards from the main deck.

## Adjustments for 1-2 Players

As an optional module, quick start cards can also make the initial setup more dynamic even in games without organizations.

Before players choose an organization and any quick start cards, reveal 4 quick start cards at random. Mark any orbiters, signals, or traces shown in the top half of those cards using neutral color pieces. Ignore the bottom half of those cards.

### Neutral Signals

When resolving a completed sector that includes neutral markers, treat those markers as if they belonged to another player.

Because there can be at most 2 neutral markers in a sector, the neutral color can never win. But if the neutral color has the second-most markers, leave one neutral marker in the sector.

## Signal Tokens

Some effects give a signal token. Signal tokens appear on some new cards and alien species. Some organizations can even gain signal tokens as income.

### Using Signal Tokens

You may use signal tokens only during a Scan action.

For each signal token you discard during a Scan action:

1. Discard 1 additional card from the card row.
2. Mark 1 signal in the sector matching that discarded card.

Because the card row is replenished only after the whole Scan action is finished, you cannot use more than 2 signal tokens during a single Scan action.

If an effect includes a Scan action, signal tokens may be used during that scan. If an effect merely marks signals but is not a Scan action, signal tokens may not be used.

## Solo With Organizations

Solo play with the expansion is based on the standard solo mode, with changes for organizations, quick start cards, and long-term objectives.

### Setup

Set up as a 2-player game. You may include neutral quick start cards as recommended in the 1-2 player adjustments.

Choose a difficulty and prepare solo mode as usual. The rival does not get an organization or quick start cards. Instead, the rival gets these alternative benefits:

1. Starting action deck: Shuffle the new extra starting action card into the rival's action deck.
2. Publicity: Set the rival's starting publicity to 4.
3. Objectives: Ignore this step when playing on 1-star difficulty.
   - Remove all level I objectives from the game.
   - Stack level II and level III objectives based on difficulty, as in the base solo rules.
   - Reveal the top 3 tiles as usual; these will be three level II objectives.
   - Shuffle the 3 long-term objective tiles and reveal 2 at random. The third is not used.

At the start of the game, there will be 5 objectives revealed: 2 long-term objectives and 3 level II objectives.

Choose your organization and quick start cards as in a multiplayer game. You make this choice after seeing the neutral quick start cards and the 5 revealed objective tiles.

### Long-Term Objective Tiles

Long-term objective tiles differ from regular objectives in two ways.

No replacement:

- When you complete all 3 tasks on a long-term objective, remove that tile from the game.
- You cannot spend a completed long-term objective to prevent rival advancement at the end of rounds.

Penalty for uncompleted tasks:

- At the end of rounds 2, 3, and 4, the rival gains 1 progress for each uncompleted task still visible on any long-term objective tile.
- Example: If 2 tasks on a long-term objective are still uncompleted at the end of the round, the rival gains 2 progress.

Uncompleted long-term objectives are not scored by the rival at the end of the game. They only cause rival progress during the game.

Tasks on long-term objectives follow the same trigger rule as standard objectives. If a single trigger could mark multiple tasks, even on different objectives, you must choose only one.

### When You Run Out of Advanced Action Cards

At higher difficulty levels, the rival may run out of available advanced action cards. If the rival should gain an extra card and no cards remain to be added, the rival gains 8 progress instead.

## Printed FAQ

### Can We Play 5 Rounds With Organizations?

Not recommended. Organizations are balanced for 4 rounds. Playing 5 rounds would also prolong the game significantly.

### Can We Play 4 Rounds With Quick Start Cards but Without Organizations?

Quick start cards do not fully compensate for the missing round. Without organizations, players will achieve less than intended and game balance is affected. If you want to try this, use the base-game starting income card in place of an organization.

### Helion Assembly and Cosmos Strategy Group

Although their topmost abilities resemble triggerable missions, they do not count as triggerable missions.

If a single trigger fulfills both a triggerable mission and one of these abilities, you can mark both.

### Futurespan Institute

You may have only 1 prepared card at a time.

A prepared card is no longer considered to be in your hand and cannot be used for any purpose other than being played for free if the Futurespan token has been reached.

The once per round ability can be used only if the Futurespan token is on the scoring track.

## Official Online FAQ Clarifications

This section supplements the PDF with rules clarifications from `docs/arch/spaceAgency/faq.md`.

### General Clarifications

- Organizations were balanced for 4 rounds. Playing 5 rounds with organizations is not recommended because it changes balance and lengthens the game.
- Quick start cards alone do not fully compensate for removing round 1. If playing 4 rounds with quick start cards but without organizations, use the base-game starting income card in place of an organization.
- When playing with organizations, leave the base-game starting income cards in the box.
- Most organizations allow multiple income increases at setup. Resolve them after quick start cards and organization starting resources. You may tuck from your initial 4 main deck cards plus any additional cards gained during setup.
- If multiple income increases are resolved sequentially, a card drawn from one income increase can be tucked for a later income increase.
- Players may agree to resolve starting income increases in player order for competitive precision. In that case, first resolve quick start cards and organization starting resources for everyone, then each player resolves all income increases in turn order.

### Organization Clarifications

1. Orbital Dynamics:
   - Once per round ability: If one of the two probes you moved was on a space with asteroids, you may add another move to get it out of the asteroid space, for example by paying 1 energy or discarding a card with 1 move in its free-action corner.

2. Fenwick Research Center:
   - Passive power: The 5-publicity technology cost is mandatory. You cannot choose to research technology for the usual 6 publicity instead.
   - Once per round ability: The gained card may come from the card row or be drawn randomly from the main deck.
   - If you take a random card from the main deck, reveal it so everyone can check its free-action corner effect.
   - A randomly drawn card may have a free-action corner effect you cannot use.

3. Stratus Core:
   - Once per round ability: Do not discard the cards in the card row.

4. Deep Sky Survey:
   - Passive power: Analyzing data with no energy cost is mandatory. You cannot choose to pay the usual 1 energy instead.
   - Once per round ability: You may exchange an alien card with a card from the card row. The alien card then enters the card row.

5. Turing Systems:
   - Once per round ability: Choosing a technology does not count as researching that technology. It does not trigger effects that require researching a technology.
   - The chosen technology only lets you use its ability for one turn.
   - If you choose the orange technology that increases your probe limit, you may launch a probe that turn even if this exceeds your usual limit.
   - You do not lose that probe when the temporary effect ends, but your normal probe limit applies again for later launches.

6. Cosmos Strategy Group:
   - Passive power: When you play a card as a main action, you may mark a space and gain the depicted reward if the card's top-right sector color matches an available space.
   - If the played card has a black sector color, choose which matching space to mark.
   - This passive power resembles a triggerable mission but does not count as one.
   - Once per round ability: It gives you a card and clears all marked spaces on the passive power, allowing them to be marked again.

7. Sentinel Probe Network:
   - Once per round ability: When you play a card as a main action, you may also gain that card's free-action corner effect.
   - Fully resolve the card first, then gain the free-action corner effect.
   - This ability cannot be used on alien cards.

8. Mission Relay:
   - You start with a specific gold scoring tile already marked.
   - The organization card shows the two sides of the same tile. Depending on which side is face up in the game, mark the lowest-scoring space on that side.
   - This mark is extra to the usual three gold tile markings per game.
   - Because the tile is already marked by you, you cannot mark it again.
   - If you mark all four gold scoring tiles, the gold box on Solvay Conference (#113) has no effect for you.
   - Once per round ability: The gained card may come from the card row or be drawn randomly from the main deck. If drawn randomly, reveal it so everyone can check its income corner.

9. Xenolab:
   - Xenolab uses three special tiles.
   - Each tile has an active side with a discounted main action and a reverse side with a specific trace.
   - All three tiles begin active side up.
   - When you use a tile's discounted action, flip that tile to its reverse side.
   - A flipped tile cannot be used again until it is flipped back.
   - To flip a tile back to active, you must gain the specific trace shown on its reverse side.
   - You still mark that trace normally; the trace is not paid or consumed.

10. Futurespan Institute:
   - The Futurespan token begins in your possession.
   - To prepare a card, place a card from your hand face up beside the organization card, then place the Futurespan token ahead of your score marker on the scoring track. The card's cost determines how many spaces ahead it is placed.
   - When your score marker reaches or passes the Futurespan token, you may play the prepared card as a main action for free.
   - You do not have to play the prepared card immediately after reaching the token.
   - When you play the prepared card, take the Futurespan token back so you can prepare another card later.
   - You may have only one prepared card at a time.
   - A prepared card is not in your hand, does not count toward your hand limit, and cannot be discarded, used for a free-action effect, or tucked for income.
   - You cannot prepare Centaurians cards because those cards cost energy to play.
   - Once per round ability: The Futurespan token must be on the scoring track. Advancing it delays when you can play the prepared card for free.

11. Helion Assembly:
   - Passive power: When you research a matching technology, you may mark spaces and gain the depicted rewards.
   - This passive power resembles a triggerable mission but does not count as one.
   - If researching a technology matches both this passive power and a triggerable mission, you may mark both.
   - Once per round ability: To gain the income increase, turn off one of your probe or telescope technologies by removing it from its slot and placing it beside your player board.
   - You may no longer use the turned-off technology's ability.
   - Even though its slot is empty, you may not research the same technology again.
   - For all other purposes, including end-game scoring, treat the turned-off technology as researched.
   - The once per round ability also clears all marked spaces on the passive power.

### Main Deck Card Clarifications

- #2 Tracking and Data Relay Satellite: The free-action corner effect refers to the card's top-left corner. Alien card free-action corners that give 2 publicity, 1 data + 1 point, or 1 move + 1 point qualify. If showing a 2-publicity corner, gain only 1 publicity.
- #5 Servicing Mission: Resolve the movement before checking whether you have a probe adjacent to Earth.
- #7 Twin Probes: The mission requires 2 probes in space at once, each on a space with a different non-Earth planet. Orbiters and landers do not count.
- #8 Quantum Data Storage: The data pool is the area on the left of your player board that holds up to 6 data. Data in your computer do not count for this effect.
- #10 First Human in Space: Publicity is capped at 10. Publicity beyond 10 is lost.
- #12 Two-planet Flyby Maneuver: Visiting a planet means moving to a space with that planet. Already being there does not count. Leaving and returning counts as one visited planet. The effect lasts for the whole turn, including later free-action moves.
- #13 James Clerk Maxwell Telescope: You do not pay the base Scan action cost, but you still pay additional costs from purple technologies. The signal token is gained only after resolving the scan, so it cannot be used during that same scan.
- #15 Iterative Engineering: If the technology tile has a printed immediate bonus, gain that immediate bonus once for acquiring the tile and once again from this card. This does not apply to extra immediate bonuses that grant a probe launch or 2 data.
- #16 Astronaut Training Experience: Resolve the main effect before checking the mission condition. If you had 0 publicity when playing the card, you cannot complete its mission immediately because you first gain 4 publicity.
- #17 Paid Media Coverage: You may play it for 0 credits, though this usually matters only for timing. Publicity beyond 10 is lost, but you still earn 2 points for each additional credit spent.
- #18 Contracted Research: You may play it when you have 0 publicity and still gain the technology.
- #19 New Assignment: Removing an orbiter from the first orbit space reopens that space and its extra 3-point reward. Other orbiters do not slide. Placing a probe on the planet does not count as launching or visiting, and does not grant visit publicity. The card may exceed your probe limit, but normal launches still obey the limit.
- #20 Well Executed Project: The free-action corner effect refers to the card's top-left corner.
- #21 Orbital Refueling: Even though it costs 0 credits, playing it for its effect is still a main action.
- #22 TESS Satellite: Even though it costs 0 credits, playing it for its effect is still a main action. Mark signals as usual, but return the replaced data to the general supply instead of gaining it.
- #23 Neutral Buoyancy Training: First-printing erratum: the triggerable mission condition should use a hexagonal action icon, meaning it triggers when you perform an orbit or land action, not merely from having an orbiter or lander.
- #26 New AI Models: Even though it costs 0 credits, playing it for its effect is still a main action.
- #27 Better Solar Panels: Even though it costs 0 credits, playing it for its effect is still a main action. Orbiters and landers are not probes in space.
- #28 Restructuring: You may choose not to discard any card.
- #29 Abandoned Mission: You may return an uncompleted conditional mission, even if its condition has already been met, or a triggerable mission with at least one unmarked space. If returning a triggerable mission with marked spaces, return those markers to your supply.
- #32 MUREP Idea Competition: You still gain 1 publicity and 2 random cards even if you have no cards left in hand after playing this card.
- #33 Delayed Launch: You cannot take free actions after you pass, so a probe launched by this mission cannot move until the next round. If you are at your probe limit when you pass, you cannot launch from the mission, but you may still mark the mission.
- #34 Private Sector Investment: The first effect is an income increase; tuck another card from your hand for income and immediately gain its income resource. For the second effect, determine which technology type you have the most of and draw 1 random card for each tile of that type.
- #35 NASA Exoplanet Archive: Data in both your data pool and your computer count toward the mission condition.
- #37 Pandora Satellite: Even though it costs 0 credits, playing it for its effect is still a main action. Mark signals as usual, but return the replaced data to the general supply instead of gaining it.
- #38 Space Rendezvous: Resolve the movement before checking whether any of your probes occupies the same space as another probe. The other probe may belong to you or an opponent.
- #42 LOFAR Array: You do not pay the base Scan action cost, but you still pay additional costs from purple technologies. There are 8 sectors; in each sector, you must either have won the sector or have at least one signal marked there.

### Alien Card Clarifications

- #ET.1: There are 7 glyph token types. The gold box gives 3 points for each token of the type you have the most of. These tokens still count for different-token sets at end-game scoring.
- #ET.3: First-printing erratum: the triggerable mission conditions should use hexagonal action icons, meaning they trigger when you perform an orbit or land action.
- #ET.9: There are 7 glyph token types. The gold box gives 1 point for each glyph in your largest set of different glyph tokens, up to 7. That set still scores normally at the end of the game.
- #ET.11 Biosignature Screening, #ET.12 Physical Characterization, and #ET.13 Genome Characterization: Marked overflow spaces below the Amoeba board count for these gold boxes.
- #ET.14 Breakthrough Theory: Marked overflow spaces below the Amoeba board count toward the mission condition. Discovery spaces do not. The reward gives points for empty spaces on the Amoeba board only; overflow spaces and rare empty discovery spaces below the board do not count.
- #ET.19 Extreme Conditions Testing: You may remove your marker from a space on the Amoeba board, a discovery space below it, or an overflow space belonging to this species. Based on that space's color, gain bonuses from all organelle tokens in the matching section and move the tokens as usual.
- #ET.34 and #ET.38: The effect may exceed your probe limit, but normal launches still obey the limit.

### Promo Card Clarification

- #SE EN 03 Space Agencies: After playing the card, launch a probe and gain a signal token. In your next game with organizations, you are dealt 1 extra quick start card but still choose only 2. This is an unusual effect that carries into the next game and should be treated as a small, optional fun advantage.
